# Mind Echoes

_Capture your thoughts, reflect on your gratitude, and explore your mind, all in one secure space._

<https://mind-echoes.netlify.app>

## Summary

This is a diary application developed with SvelteKit and SQLite. Entries are called "echoes".

This project showcases in particular how to implement various account features by hand (email verification, password reset, device management).
